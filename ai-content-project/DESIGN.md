# DESIGN.md

## 气质与意象

专业内容生产工坊 — 像一间光线充足的创意工作室，桌上铺着深色哑光台面，墙上挂着刚输出的海报样稿，显示器里视频正在播放。整体感觉是"有序的创造力"：干净、高效，但不冷漠。

## 配色方案

- 主背景：#F5F5F7（微灰暖白，像工作室白墙）
- 卡片背景：#FFFFFF（纯白，悬浮于主背景上）
- 主操作色：#1D4ED8（沉稳蓝，用于主按钮与关键交互）
- 次操作色：#6366F1（靛紫，用于辅助按钮与生成类型标识）
- 文字主色：#18181B（近黑，标题与正文）
- 文字次色：#71717A（中灰，标签与说明）
- 分隔线：#E4E4E7（浅灰，区域分隔）
- 警示色：#F59E0B（琥珀，用于 ⚠️ 提示信息）
- 成功色：#10B981（翠绿，用于完成状态）
- 内容源标识色：链接抓取 #3B82F6 / 文件识别 #8B5CF6 / AI生成 #EC4899 / 人工粘贴 #F97316
- 生成类型色：文章 #10B981 / 海报 #6366F1 / 视频 #F59E0B
- 渠道标识：CMS官网 #1D4ED8 / 公众号 #07C160 / 视频号 #FA9D3B / 抖音 #161823 / 小红书 #FE2C55

## 字体排版

- 字体族：PingFang SC, Microsoft YaHei, system-ui（复用 globals.css 配置）
- 标题：font-semibold, tracking-tight
- 正文：font-normal, leading-relaxed
- 标签/参数：text-sm, text-muted-foreground

## 动效与交互

- 按钮悬停：微上移 + 阴影加深（translate-y -0.5px, shadow-md → shadow-lg）
- 卡片悬停：阴影变化（shadow-sm → shadow-md）
- 状态切换：opacity 过渡 200ms
- 视频/海报预览区域：aspect-ratio 锁定，防止布局偏移
- 页面切换：使用 Next.js App Router 原生转场

## 页面结构

- 全局顶部导航：Logo + 系统名 + 用户头像
- 单栏居中布局（max-w-5xl），内容纵向堆叠
- Dashboard：数据统计卡 + 内容池列表 + 快捷创建入口
- 内容创建页：4 Tab 切换（链接/文件/AI/粘贴）
- 海报编辑器：左栏表单 + 右栏实时预览
- 结果页：预览区 + 操作按钮 + 分发渠道

## 设计禁忌

- 不要用大面积渐变色块做背景
- 不要用圆角过大的按钮（保持 radius-md）
- 不要在同一个区域放置超过 3 种颜色
- 不要用 emoji 代替图标（除原始设计稿中已有的 emoji 标识）
- 不要使用深色主题（本产品面向内容运营人员，浅色更友好）
